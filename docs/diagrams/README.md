# Diagramas técnicos

Esta carpeta contiene diagramas mínimos y alineados con el estado real del proyecto.

Formato elegido:

- `Mermaid (.mmd)`

Motivo:

- se versiona bien en Git
- es fácil de editar
- GitHub puede renderizarlo directamente en muchos contextos
- evita depender de binarios o herramientas gráficas externas

Archivos incluidos:

- `architecture.mmd`: diagrama de arquitectura/componentes
- `sequence-login.mmd`: secuencia del flujo de login
- `der.mmd`: DER básico del modelo de datos implementado
